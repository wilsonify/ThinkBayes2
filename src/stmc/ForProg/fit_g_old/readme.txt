
The input data have to be on fort.10 in the format given by the
example file.                =======

Compile and run gfit.f

Plot with gnuplot gfit.plt.

Inclusion of alternative subg_fln.f files allow different types of fits.

Data sets: (*.D files)
=========

berg:        Berg-Neuhaus PRL 68 (1992) 9 data for the 
             surface tension, 1/x (1ox) fit (x=L).

beta_cd:     Berg-Neuhaus PRL 68 (1992) 9 beta_c data, 1/x fit x=V=L^2.

potts_10qbc: 2d 10-state Potts model beta_c(L) data from equal heights.

	     Used in the example for 1/N fit.
