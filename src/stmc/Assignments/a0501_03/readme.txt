
 
How to create figure  fig_2dI_es:
=================================

The gnuplot driver file is es.plt.
                           ======

1. Run ferdinand.f (here) to generate ferdi.d

2. Get the data file mu2d02q02.dat from a0501_01 
   after running p_mu.f there.

3. Run ana_pmu.f which generates p2d02q020.d.

4. gnuplot es.plt.

5. C.plt   gives the specific heat.

6. F2q.plt gives the free energy.

7. S2q.plt gives the entropy.
